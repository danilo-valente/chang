Chang
=====

Git Changelog generator, based on Angular's commit message notation

## Installation

```sh
npm install -g chang
```

## Usage

```sh
chang [-r <release_name>] [-o <output_file>] [-g <github_repo>] [-l <language]> [-t <starting_tag>]
```

Or

```sh
chang -h
```

To display help.
